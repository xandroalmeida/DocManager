h1. Document Manager

h2. Português

Este é um simples gerenciador de documentação. Eu o uso para gerar os diferentes tipos de documentos de um projeto de software.
Este sistema permite você organizar e editar seus documentos usando alguma linguagem de markup e visualiza-lo em formato HTML pelo browser.

Hoje a edição deve ser feita diretamente no file ssystem (configurado no contex-param "base_document_directory", que deve ter a seguinte estrutura
Projeto 
 +-Modulos
    +documento.(md|wiki|textile)

Atualmente apenas são aceitos as seguintes linguages de markup

* Textile
* Markup
* MediaWiki
